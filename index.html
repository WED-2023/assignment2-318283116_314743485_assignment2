Game Instructions

  • Use the arrow keys (←, →, ↑, ↓) to move your spaceship within the bottom 40% of the screen.
  • Press your configured firing key (Space or a selected letter key) to shoot bullets upward.
  • Destroy all enemy spaceships before time runs out or before losing all lives.
  • Enemies move horizontally across the screen, changing direction when hitting screen edges, and shifting downward each time.
  • Enemies shoot bullets randomly — avoid enemy fire!
  • Scoring system based on enemy row:
    • Top row – 20 points
    • Second row – 15 points
    • Third row – 10 points
    • Bottom row – 5 points
  • You have 3 lives. Each hit by an enemy bullet costs one life. When all lives are lost, the game ends.
  • Enemy and bullet speed increases every 5 seconds, up to 4 times, making the game progressively harder.
  • Game over conditions:
    • Victory: All enemies destroyed.
    • Defeat: All lives lost.
    • Time expired:
      • If the player scored less than 100 points — "You can do better!" message appears.
      • If the player scored 100 points or more — "Winner!" message appears.
  • After the game ends, a personal leaderboard showing the top scores is displayed.
  • You can start a new game anytime by clicking the "New Game" button.

Additional Features Implemented

  • Custom spaceship color selection
  • Game history system with leaderboard
  • Sound effects and background music
  • Visual effects during speed acceleration

Development Challenges

  • Implementing an accurate collision detection system between bullets and spaceships while maintaining performance and avoiding false positives.
  • Managing smooth and responsive player movement while handling multiple simultaneous keyboard inputs.
  • Synchronizing enemy behavior and their speed increases over time to maintain balanced difficulty.
  • Designing a dynamic game configuration system (controls, colors, sounds, game time) that seamlessly integrates into gameplay.
